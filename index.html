Anmälan Seminarier

Bättre Webb

Datum: 20-22 Okt, 2023

Plats: Hotell Belloras, centrala Göteborg

Konferensen:

Välkommen till "Bättre Webb för Frontendutvecklare" en exklusiv konferens skapad av och för frontendutvecklare. Vår målsättning är att fördjupa din kompetens och inspirera dig att leverera exceptionella användarupplevelser genom webbutveckling. Denna konferens är en unik möjlighet att dyka djupare in i de senaste teknikerna och bästa praxis inom frontendutveckling.

Bilder från de fina hotellet

Vad du kan förvänta dig:

  • Experttalare: Lyssna på ledande frontendutvecklare, teknikinnovatörer och UX-designers som delar sina erfarenheter och insikter.
  • Escape Room seminarier : Dessa seminarieidéer kombinerar interaktivitet och problem lösningsförmåga med frontendutvecklingsutmaningar.
  • Praktiska Workshops: Delta i hands-on workshops där du kan fördjupa dig i verkliga projekt och förbättra dina praktiska färdigheter.
  • Frontend Trender: Få en förhandsvisning av de senaste trenderna inom frontendutveckling och hur du kan implementera dem i dina projekt.
  • Nätverkande: Skapa värdefulla kontakter och utbyt idéer med frontendutvecklare från olika organisationer

Gör dig redo att förbättra dina frontendutvecklingsfärdigheter och skapa en bättre webb. Vi ser fram emot att välkomna dig till en konferens som kommer att hjälpa dig excel inom frontendutveckling och skapa webbsidor som imponerar och engagerar användarna.

De olika seminarierna

Fredag:

Escape the Broken Website Code:

Deltagarna måste använda sina frontendutvecklingsfärdigheter för att rätta till buggar och problem på en trasig webbplats.

Frontend Puzzles and CSS Challenges:

Ett seminarium som innehåller gåtor och CSS-utmaningar som deltagarna måste lösa för att fortsätta genom rummen.

JavaScript Escape Room,The Quest for Interactivity

Deltagarna måste använda JavaScript för att skapa interaktiva funktioner och lösa gåtor för att fly från rummet

Lördag:

Responsive Design:

Ett seminarium där deltagarna använder sina frontendkunskaper för att göra webbplatsen responsiv och anpassad till olika enheter och skärmstorlekar.

Web Accessibility:

Deltagarna måste använda webbtillgänglighetsprinciper och tekniker för att göra webbplatsen användarvänlig för alla.

Frontend Performance Race:

Ett seminarium som fokuserar på att förbättra webbplatsens prestanda genom att optimera frontendkoden.

Design Patterns and Escape Room Puzzles:

Deltagarna måste använda designmönster och bästa praxis inom frontendutveckling för att lösa gåtor och problem.

Web Animation Mysteries:

Ett seminarium som inkluderar uppgifter med webbanimationer som deltagarna måste skapa och anpassa.

Söndag:

E-commerce Frontend Challenge:

Deltagarna måste använda sina frontendutvecklingsfärdigheter för att förbättra användarupplevelsen och konverteringsgraden på en e-handelswebbplats.

Building a Progressive Web App:

Ett seminarium där deltagarna måste bygga och optimera en progressiv webbapp genom att lösa uppgifter och gåtor.
